# Privacy Policy for GrammarWise

**Last Updated:** November 11, 2024

## Overview
GrammarWise is a Chrome extension that helps users check grammar and translate text using the Anthropic Claude API.

## Data Collection and Usage

### What We Collect
GrammarWise does NOT collect, store, or transmit any personal data to our servers. The extension operates entirely within your browser and directly communicates with Anthropic's API.

The only data stored is:
- **API Key**: Your Anthropic Claude API key is stored locally in your browser using Chrome's sync storage
- **User Preferences**: Your default tone, language, and translation settings are stored locally

### How Your Data is Used
- **Selected Text**: When you use the grammar checker or translation features, the selected text is sent directly to Anthropic's API for processing
- **API Communications**: All communication happens directly between your browser and Anthropic's servers using your personal API key
- **No Third-Party Tracking**: We do not use any analytics, tracking, or advertising services

### Data Storage
- All data is stored locally in your browser using Chrome's storage API
- Your API key is stored securely in Chrome's sync storage
- We do not have access to your API key or any data you process

### Third-Party Services
GrammarWise uses the Anthropic Claude API to provide grammar checking and translation services. When you use these features:
- Your selected text is sent to Anthropic's servers
- Anthropic's privacy policy applies to this data: https://www.anthropic.com/privacy
- You are responsible for your own API key and usage

### Data Security
- Your API key is stored using Chrome's built-in secure storage
- All API communications use HTTPS encryption
- We do not transmit your data to any servers except Anthropic's API

### User Rights
You can:
- Delete your API key and preferences at any time by uninstalling the extension
- Clear your stored data from the extension settings
- Stop using the extension at any time

### Changes to This Policy
We may update this privacy policy from time to time. Any changes will be reflected in the extension's repository and store listing.

### Contact
For questions or concerns about this privacy policy, please open an issue on our GitHub repository.

## Compliance
This extension complies with:
- Chrome Web Store policies
- GDPR requirements (no personal data collection)
- Anthropic's API terms of service
