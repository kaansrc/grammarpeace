## Detailed Description (Max 16,000 characters)

Transform your writing with GrammarWise, an AI-powered grammar checker and translator that works on any website!

**Features:**

✅ **Multi-Language Grammar Checking**
- Fix grammar, spelling, and punctuation in 15+ languages
- Auto-detect language or choose from English, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Korean, Chinese, Arabic, Hindi, Turkish, Dutch, and Polish
- Smart corrections that preserve your writing style

✅ **Tone Adjustment**
- Professional: Business-appropriate writing
- Casual: Conversational and relaxed
- Friendly: Warm and approachable
- Formal: Academic and official
- Concise: Brief and to the point

✅ **Translation Feature**
- Translate text between 15+ languages
- Auto-detect source language
- Quick access through tabbed interface
- Same copy/replace functionality

✅ **Easy to Use**
- Select any text on any webpage
- Click the floating button
- Get instant corrections or translations
- One-click replace or copy to clipboard

✅ **Works Everywhere**
- Gmail, Slack, Discord, Twitter/X
- Google Docs, Notion, Obsidian
- Reddit, LinkedIn, Facebook
- Text editors and forums
- Any text input field

✅ **Privacy-Focused**
- Uses your own Claude API key
- No data collection or tracking
- All processing happens through Anthropic's secure API
- Your data is never stored on our servers

✅ **Customizable**
- Set default tone and language preferences
- Configure translation defaults
- Right-click context menu support
- Clean, modern interface

**How to Use:**

1. Get your API key from Anthropic (https://console.anthropic.com/)
2. Install GrammarWise
3. Enter your API key in extension settings
4. Select text on any website
5. Click the purple button that appears
6. Choose "Fix Grammar" or "Translate"
7. Get instant results powered by Claude AI

Unlike other grammar checkers, GrammarWise:
- Uses state-of-the-art AI (Claude Haiku 4.5)
- Supports 15+ languages with auto-detection
- Includes translation features
- Respects your privacy (no data collection)
- Works on any website
- Lets you control costs with your own API key
